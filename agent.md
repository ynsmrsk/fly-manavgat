# Fly Manavgat Agent Guide

## Project Scope

- Build a single page website
- Keep all sections inside `index.html`
- Put all authored styles in `style.css`
- Put all authored JavaScript in `script.js`

## Markup Rules

- Write minimal semantic HTML
- Keep the structure clean and easy to extend
- Avoid placeholder complexity until content is defined

## CSS Rules

- Write scalable, reusable, clean minimal CSS
- Prefer shared tokens and low-specificity selectors
- Start simple and expand section by section

## JavaScript Rules

- Keep JavaScript minimal
- Avoid unnecessary libraries
- If a library is needed, prefer one small focused dependency
- Do not use semicolons at the end of JavaScript lines

## Brand Context

- Website is for a paraglide business
- Service area is Manavgat and Alanya
- Current first task is the global sky gradient background
- Gradient direction is dark blue at the top to light blue at the bottom
- The gradient should stay fixed while the user scrolls

## Current Baseline

- Minimal semantic page scaffold is in place
- Fixed full-page sky gradient is implemented in `style.css`
- Content and visual language will be added incrementally
